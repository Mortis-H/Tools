## Timeline Discussion Canvas

Path:

```text
timeline-discussion-canvas/index.html
```

GitHub Pages URL:

```text
https://mortis-h.github.io/Tools/timeline-discussion-canvas/
```

This is a self-contained front-end tool for sketching timeline discussions:

- add lanes
- add duration blocks
- add instant signal markers
- add guide lines
- drag items along the X axis
- edit labels, colors, notes, and timing
- export/import JSON manually

The tool does not automatically read or write local files. Use `Export JSON` to download a canvas state, and `Import JSON` to load it back later.
