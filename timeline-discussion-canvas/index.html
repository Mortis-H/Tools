Timeline Discussion Canvas

A schematic timeline for CPFW / AM / silicon discussions. Keep ordering, duration, and guide marks in one canvas without turning rough discussion notes into fake precision.
Local autosave Relative time Undo / redo Dark / light
Inspector Select a lane, block, signal, or guide.
Local only

Blank canvas

Use Add lane, then Add block or Add signal. Click the canvas to place new items. Drag items along X to adjust their position on the relative-time axis.

Object types

  • Block: duration, with start and end.
  • Signal: instant event, shown as a diamond and vertical tick.
  • Lane: Y-axis category. Rename or reorder it from the inspector.

Fast workflow

  • Pick an add mode, then click the canvas.
  • Select an object, press Ctrl+C, then click the canvas to place a snapped copy.
  • Drag the canvas to pan; use the wheel to zoom time around the pointer.
  • Export JSON when the discussion state should travel with the note.
