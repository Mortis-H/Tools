Timeline Discussion Canvas

A blank, editable timeline canvas for discussions. Add lanes, duration blocks, and instant signal markers. The X axis is relative schematic time: use it to show ordering and approximate length, not measured cycles unless you explicitly decide to enter measured values. The Time range box only changes the width of this schematic axis. Version: 2026-06-26-1101.

Blank canvas

Use Add lane, then Add block or Add signal. Click the canvas to place new items. Drag items along X to adjust their position on the relative-time axis.

Object types

  • Block: duration, with start and end.
  • Signal: instant event, shown as a diamond and vertical tick.
  • Lane: Y-axis category. Rename or reorder it from the inspector.
